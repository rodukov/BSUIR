# Запросы
- StudentsR1 (Вывести студентов, их курсы и дату регистрации) - Выборка
- StudentsR2 (Количество регистраций по курсам по годам) - Кросс-табличный запрос
- StudentsR3 (Обновление базы данных на 90ч, и назад на 310ч) - Обновление
- StudentsR4 (Добавить запись регистрации 2009г) - Добавление
- StudentsR5 (Удалить регистрации до 2010 года) - Удаление
- StudentsRT6 (Создать таблицу студентов младше 2006г) - Создание таблицы + Drop
## Отчетный запрос
- StudentsReport (студенты с курсами (все данные)) - Для отчета

# Формы
- StudentsF1 (форма с данными студента)
- StudentsF2 (форма с данными студента и зарегистрированными курсами) + REGF2 (зависим. форма)

# Отчеты
- Courses (Курсы и какие ФИ на них есть)
- StudentsReport (Полная информация о студентах и их курсах)

# Макросы
- Students M1 (F2 младше 2006 года)
- Run M1 (корректный запуск Students M1)
- Quit (корректный выход из Access)
- AutoExec (макрос автозапуска)
